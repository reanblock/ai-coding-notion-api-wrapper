# Notion API Wrapper

A simple API Wrapper for Notion based on [this tutorial](https://www.youtube.com/watch?v=QlUt06XLbJE).

## Install

## Run

Run using `npm`:

```bash
npm run cli getPage
```

Run using `node` and `jq` (to get pretty formatted JSON in the terminal!)

```bash
node index.js getPage | jq
```
